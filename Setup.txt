Setup
create user helium24, password proton2
nano /etc/supervisor/conf.d/helium24.conf
-> Copy the following
>>
[program:helium24]
command=/usr/bin/dotnet ./WedSite.dll --server.urls "http://*:80"
user=root
stderr_logfile = /home/helium24/netcoreapp3.1/stderr.log
stdout_logfile = /home/helium24/netcoreapp3.1/stdout.log
directory = /home/helium24/netcoreapp3.1
>>
supervisorctl reread
supervisorctl update
supervisorctl start helium24
